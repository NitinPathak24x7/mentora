**Mentora**

**Digital Mental Health & Student Wellness Platform**

Mentora is a web-based platform designed to provide accessible mental health support for students. The platform aims to bridge the gap between students and mental wellness resources through self-help tools, educational content, and an intuitive user experience.


**Features**

- Mental health awareness content
- Interactive flashcards for learning and self-reflection
- Self-help resources and wellness guides
- Responsive and modern user interface
- Easy navigation and accessibility-focused design
- Student-friendly dashboard experience


## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Data Storage
- JSON-based data handling
