# Custom Relaxing Images

Place your relaxing images in this folder.

Recommended filenames so the gallery auto-picks them:
- user_1.jpg
- user_2.jpg
- user_3.jpg
- user_4.jpg
- user_5.jpg
- user_6.jpg

Notes:
- .png or .jpeg are fine too (keep the same names).
- If fewer than 6 are present, the site will fall back to the default SVGs.
